# claude-ultracode-review-public

Public workspace for subscription-backed Claude Code review; no research or patient data.

采用「三方审查」协作模式：Claude Code 负责规划与深度审查（多智能体交叉验证 + 安全审查 + 科研级核查），ChatGPT 桌面 app（Work 模式）负责执行，Codex review 与 GPT-5.6 Pro 提供另外两份独立审查。完整流程见 [WORKFLOW.md](WORKFLOW.md)，每个任务的计划放在 [plans/](plans/) 目录。
